First, a brief introduction to what Markdown is:

Markdown is a lightweight markup language that allows people to write documents in plain text formats that are easy to read and write.

Markdown was created by John Gruber in 2004.

Markdown writes documents that can export HTML, Word, images, PDF, and Epub

Documents in various formats.

Markdown writes the document suffix.md,.markdown.
